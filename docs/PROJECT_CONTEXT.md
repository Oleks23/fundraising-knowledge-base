# Fundraising Command Centre

Project Context Document

This repository contains architecture, implementation, and operational documentation for the Fundraising Command Centre (FCC).

The project is based on a universal fundraising operations model:

- Initiatives
- Activities
- Dependencies
- Commitments
- Risks
- Knowledge

Primary implementation platform:
- Microsoft 365
- SharePoint Lists
- Power Automate
- Power BI
- Microsoft Teams

Primary CRM target:
- Raiser's Edge NXT

Status:
- Initial repository setup
